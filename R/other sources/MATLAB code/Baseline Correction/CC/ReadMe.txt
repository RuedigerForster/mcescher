

This is a package for performing corner-cutting and Bezier smoothing for
baseline correction. 

The main function is "cornerCuttingBezierCurve"

Inputs: 

[x,y] a set of paired variables indicating the coordinates of the signal points. 

Steps:  Numbers of the corner cutting iterations. 



OutPuts: 

bezierCurve:  The smoothed baseline found using the CC-Bezier thechnique.
